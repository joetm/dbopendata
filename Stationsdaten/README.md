Stationsdaten der Deutschen Bahn
================================

STAND: März 2016
Quelle: http://data.deutschebahn.com/datasets/stationsdaten/
Lizenz: Creative Commons Attribution 4.0 International (CC BY 4.0)

Die Stationsdaten enthalten eine Liste der Bahnhöfe inkl. Adressen, Aufgabenträger und Link zu Betriebsstellen.

In den veröffentlichten Daten sind zu jedem Bahnhof folgende Informationen enthalten:

* Bundesland: Bundesland, z.B. Hessen.
* BM: Bahnhofsmanagement, z.B. Darmstadt.
* Bf.Nr.: Eindeutige Nr. des Bahnhofs, z.B. 119.
* Station: Name der Station, z.B. Altheim (Hess).
* Bf DS 100 Abk.: Verweis auf Betriebsstelle, z.B. FAT.
* Kat. Vst.: Bahnhofskategorie, z.B. 6.
* Straße: Postalische Adressinfo.
* PLZ: Postalische Adressinfo.
* Ort: Postalische Adressinfo.
* Aufgabenträger: Name des Aufgabenträgers, z.B. Rhein-Main-Verb.
* Verkehrsverb.: Abk. des Aufgabenträgers, z.B. RMV.
* Fernverkehr: Fernverkehrszug hält an dieser Station: ja/nein.
* Nahverkehr: Nahverkehrszug hält an dieser Station: ja/nein.
