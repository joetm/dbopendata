Stationsdaten (RNI) der Deutschen Bahn
======================================

STAND: Januar 2016
Quelle: http://data.deutschebahn.com/datasets/stationsdaten_regio/
Lizenz: Creative Commons Attribution 4.0 International (CC BY 4.0)

Die Stationsdaten enthalten eine Liste der Bahnhöfe von DB RegioNetz Infrastruktur GmbH inkl. Aufgabenträger.

In den veröffentlichten Daten sind zu jedem Bahnhof folgende Informationen enthalten:

* Bundesland: Bundesland, z. B. Hessen.
* Regionalbereich: Bahnhofsmanagement, z. B. Kurhessenbahn (KHB).
* Bf.Nr.: Eindeutige Nr. des Bahnhofs, z. B. 8263.
* Station: Name der Station, z. B. Ahnatal-Casselbreite.
* Bf DS 100 Abk.: Verweis auf Betriebsstelle, z. B. FATC.
* Kat. Vst.: Bahnhofskategorie, z. B. 6.
* Straße: Postalische Adressinfo.
* PLZ: Postalische Adressinfo.
* Ort: Postalische Adressinfo.
* Aufgabenträger: Name des Aufgabenträgers, z. B. Nordhessischer Verkehrsverbund.
* Verkehrsverb.: Abk. des Aufgabenträgers, z. B. NVV.
* Fernverkehr: Fernverkehrszug hält an dieser Station: ja/nein.
* Nahverkehr: Nahverkehrszug hält an dieser Station: ja/nein.
